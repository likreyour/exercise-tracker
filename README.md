# 🏃 运动记录App

一个轻量级的PWA运动记录应用，支持步数统计和GPS跑步追踪。

## ✨ 功能

### 核心功能
- **📱 步数统计** - 使用手机加速度传感器实时计步
- **🏃 GPS跑步追踪** - 记录跑步轨迹、距离、配速
- **📊 数据统计** - 今日/周数据可视化
- **🎯 目标设定** - 自定义每日步数和跑步目标

### PWA特性
- 可添加到手机主屏幕（像原生App一样）
- 支持离线使用
- 后台自动保存数据

## 🚀 使用方法

### 桌面端
直接用浏览器打开 `index.html` 即可（步数为模拟数据）

### 移动端
1. 用手机浏览器打开
2. 点击"添加到主屏幕"
3. 即可像原生App一样使用

## 🛠️ 技术栈

| 技术 | 说明 |
|------|------|
| HTML5/CSS3 | 页面结构与样式 |
| JavaScript ES6 | 应用逻辑 |
| LocalStorage | 本地数据存储 |
| DeviceMotion API | 步数检测（手机） |
| Geolocation API | GPS定位 |
| Canvas API | 跑步轨迹绘制 |
| Service Worker | PWA离线支持 |

## ⚙️ 设置说明

- **每日步数目标**: 默认10,000步
- **跑步目标**: 默认5公里
- **步长**: 默认70厘米（用于估算行走距离）

所有数据保存在浏览器本地，不会上传到任何服务器。
